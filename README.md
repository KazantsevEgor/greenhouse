# Автоматизированная система управления теплицей

Учебный прототип системы для гидропонной теплицы по ТЗ №52. Проект содержит FastAPI backend, React/Vite frontend, случайную эмуляцию датчиков, цикл выращивания и автоматическое управление устройствами.

## Возможности

- запуск и остановка цикла выращивания;
- графическое представление теплицы с датчиками и устройствами;
- датчики температуры воздуха и воды, влажности, pH, освещенности и концентрации питательных веществ;
- устройства: обогреватели, кондиционеры, увлажнители, источники света, дозаторы удобрений и питательного раствора;
- автоматическое управление температурой, влажностью, pH, питательными веществами и светом;
- добавление датчиков и устройств через интерфейс;
- настройка базового плана выращивания через интерфейс;
- обновление состояния через WebSocket.

## Запуск backend

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

API будет доступен на `http://localhost:8000`, документация Swagger - на `http://localhost:8000/docs`.

## Запуск frontend

```bash
cd frontend
npm install
npm run dev
```

Интерфейс будет доступен на `http://localhost:5173`. Vite проксирует `/api` и `/ws` на backend.

## Проверка

```bash
python -m compileall app
cd frontend
npm run build
```
